# Personal Portfolio Website 🚀

A modern, responsive single-page personal portfolio built for a college academic submission. The site showcases my current education, technical skills, projects, and recent professional experiences.

## 🔗 Live Link
You can view the live deployed website here: 
👉 [https://barsa-dot.github.io/Portfolio-/](https://barsa-dot.github.io/Portfolio-/)

## 🛠️ Built With
* **HTML5** - Semantic structural layout
* **Tailwind CSS** - Modern, utility-first styling (via CDN)
* **FontAwesome** - Clean, scalable developer icons

## 📂 Highlights Included
* **Education:** Coursework details at Reva University, Bengaluru.
* **Experience:** Highlights of content development and AI/ML internship tracking.
* **Featured Project:** Hypoxia vs Control RNA-Seq Analysis using Python.
* **Community:** On-ground volunteer coordination records for regional tech events.
